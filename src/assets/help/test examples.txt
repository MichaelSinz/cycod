CHATX TEST EXAMPLES

  Examples of using the chatx test commands.

LIST EXAMPLES

  List all tests:
    chatx test list

  List tests from a specific file:
    chatx test list --file tests/mytest.yaml

  List tests from multiple files:
    chatx test list --files tests/test1.yaml tests/test2.yaml

  List tests from a directory using a glob pattern:
    chatx test list --files tests/**/*.yaml

  List tests containing specific text:
    chatx test list --contains setup integration

  List tests excluding specific text:
    chatx test list --remove skip deprecated

  List tests with multiple filters:
    chatx test list --tests setup nightly --contains java --remove skip

RUN EXAMPLES

  Run all tests:
    chatx test run

  Run tests from a specific file:
    chatx test run --file tests/mytest.yaml

  Run tests from multiple files:
    chatx test run --files tests/test1.yaml tests/test2.yaml

  Run tests from a directory using a glob pattern:
    chatx test run --files tests/**/*.yaml

  Run tests containing specific text:
    chatx test run --contains setup integration

  Run tests excluding specific text:
    chatx test run --remove skip deprecated

  Run tests with multiple filters:
    chatx test run --tests setup nightly --contains java --remove skip

  Run tests with custom output file:
    chatx test run --output-file my-test-results.trx

  Run tests with custom output format:
    chatx test run --output-format junit
