CHATX TEST LIST

  The chatx test list command lists CLI YAML tests.

USAGE: chatx test list [...]

  FILES
    --file FILE
    --files FILE1 [FILE2 [...]]
    --files PATTERN1 [PATTERN2 [...]]

  TESTS
    --test TEXT
    --tests TEXT1 [TEXT2 [...]]

  FILTERING
    --contains TEXT1 [TEXT2 [...]]
    --remove TEXT1 [TEXT2 [...]]

EXAMPLES

  EXAMPLE 1: Lists all available tests

    chatx test list

  EXAMPLE 2: Lists tests from files under current directory, that contain 'setup' or 'nightly', and 'java', but not 'skip'

    chatx test list --tests setup nightly --contains java --remove skip

  EXAMPLE 3: Lists tests from files under 'tests' directory, that contain 'test3', but not 'skip'

    chatx test list --files ../tests/**/*.yaml --contains test3 --remove skip
