Using CHATX, you can:

  - Chat with an AI model (interactively/programmatically)
  - Provide one or more inputs to the AI model
  - Use built-in function tools to:
    - Read and write files
    - Execute shell commands/scripts
    - Use MCP (Model Context Protocol) tools

USAGE: chatx <command> [...]

COMMANDS

  chatx [...]             (aka: chatx chat)
  chatx chat [...]        (see: chatx help chat)

  chatx config [...]      (see: chatx help config)
  chatx github login      (see: chatx help github login)

  chatx mcp [...]         (see: chatx help mcp)
  chatx alias [...]       (see: chatx help alias)
  chatx prompt [...]      (see: chatx help prompt)
